# Prompt Engineering Guide

This folder contains prompt templates and strategies for various AI tools.

## Structure
- `gemini/`: Narrative, script, and logic prompts.
- `midjourney/`: Visual asset and character consistency prompts.
- `veo/`: Video generation and motion control prompts.

## Tips for High-Fidelity Outputs
1. **Be Descriptive**: Use sensory language.
2. **Context is King**: Always provide the background of the "Benim TEKNOFEST'im" theme.
3. **Iterative Refinement**: Use the feedback loop to improve outputs.
